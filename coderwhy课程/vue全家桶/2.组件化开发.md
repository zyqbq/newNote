### 局部组件

更加常用

使用局部组件，`必须在父组件中注册`

使用components:{组件名:组件}在组件中注册局部组件

### 局部组件

```js
<template>
    //使用组件
  <HelloWorld />
</template>

<script>
 //引入组件
import HelloWorld from "./components/HelloWorld.vue";

export default {
  name: "App",
  components: {
     // 注册组件
    HelloWorld,
  },
};
</script>

<style></style>
```
