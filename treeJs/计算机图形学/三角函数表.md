sin 对边比斜边

cos 临边比邻边

tan 对边比邻边

**正弦函数（Sine）：**

- sin(0°) = 0
- sin(30°) = 0.5
- sin(45°) = √2 / 2 ≈ 0.7071
- sin(60°) = √3 / 2 ≈ 0.8660
- sin(90°) = 1
- sin(180°) = 0
- sin(270°) = -1
- sin(360°) = 0

**余弦函数（Cosine）：**

- cos(0°) = 1
- cos(30°) = √3 / 2 ≈ 0.8660
- cos(45°) = √2 / 2 ≈ 0.7071
- cos(60°) = 0.5
- cos(90°) = 0
- cos(180°) = -1
- cos(270°) = 0
- cos(360°) = 1

**正切函数（Tangent）：**

- tan(0°) = 0
- tan(30°) ≈ 0.5774
- tan(45°) = 1
- tan(60°) ≈ 1.7321
- tan(90°) → 无穷大
- tan(180°) = 0
- tan(270°) → 无穷大
- tan(360°) = 0
